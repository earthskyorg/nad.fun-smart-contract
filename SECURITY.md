# Security Policy

We take the security of the Nad.fun Monad contracts seriously. Please follow the
guidelines below to report vulnerabilities responsibly.

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| main    | ✅                 |
| tags    | ✅ (latest only)   |

## Reporting a vulnerability

- Email: `security@earthsky.org` (preferred)
- Telegram: [Tech Genie](https://t.me/opensea712)

Please include:

- A description of the issue and potential impact
- Steps to reproduce or proof-of-concept code
- Suggested remediation ideas (if available)

We will acknowledge receipt within **48 hours**, provide an initial assessment in
**5 business days**, and aim to ship a fix or mitigation within **14 days**.

## Safe harbor

If you follow this policy, we will not pursue legal action. Do not exploit the
vulnerability (beyond what is necessary to demonstrate it) and refrain from
public disclosure until we coordinate a fix.
